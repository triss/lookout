How this works

Motion clips, stored locally. lookout watches a fixed view on this device. When sustained motion is detected, it saves a short video clip with configurable seconds before and after the event.

Nearby motion events can be grouped into one clip. For example, if somebody opens a gate, walks through, and returns within the event-series threshold, those events can become a single clip instead of several separate files.

  1. Set the phone somewhere stable with a clear view of the doorway, yard, gate, shed, hallway, or room.
  2. Press Start camera. The page asks for fullscreen, starts the camera, and begins keeping a rolling pre-event buffer.
  3. Do not move or bump the phone while it is watching. If the view changes, pause and restart so the detector settles again.
  4. Use Clips to review saved videos. Use Share data only when you deliberately want to export observations or clips.

Fix common problems in Settings. If shadows, leaves, reflections, or rain create too many clips, raise Sensitivity threshold, Minimum size, or Minimum motion duration. If one activity is split into too many clips, raise Event series threshold. If clips miss useful context, raise Seconds before event or Seconds after event. If the phone is slow, lower Target FPS or Resolution.

Video clips may contain identifying imagery. They stay in browser storage on this device and are only included when you explicitly share clips or a bundle.
